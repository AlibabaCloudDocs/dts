# 删除消费组

调用DeleteConsumerGroup接口删除订阅通道中的消费组。

## 调试

[您可以在OpenAPI Explorer中直接运行该接口，免去您计算签名的困扰。运行成功后，OpenAPI Explorer可以自动生成SDK代码示例。](https://api.aliyun.com/#product=Dts&api=DeleteConsumerGroup&type=RPC&version=2020-01-01)
